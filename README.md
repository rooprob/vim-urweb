# vim-urweb

vim-urweb is a plugin for ur/web development in vim.

# Features

* ML syntax highlighting, thanks to vim-sml
* XML syntax highlighting, thanks to vim-xml

# Experimental Features

* UR syntax highlighting
* UR development workflow

# Bugs
